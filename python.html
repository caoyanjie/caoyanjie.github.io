【Python协程】

标签：
coroutine
asyncio
async
await
协程的并发优于多线程的并发是：协程有极高的运行效率，不需要考虑线程间同步、死锁等问题
1.普通函数
{% highlight python %} import time def task1(): for i in range(10): print(i) time.sleep(2) def task2(): chars = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j'] for i in chars: print(i) time.sleep(2) start = time.time() task1() task2() end = time.time() print(f'运行完毕，耗时: {end-start}') {% endhighlight %}
2.协程并发
{% highlight python %} import asyncio async def coroutine_task1(): for i in range(10): print(i) await asyncio.sleep(2) async def coroutine_task2(): chars = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j'] for i in chars: print(i) await asyncio.sleep(2) task1 = asyncio.create_task(coroutine_task1()) task2 = asyncio.create_task(coroutine_task2()) start = time.time() await task1 await task2 end = time.time() print(f'运行完毕，耗时: {end-start}') {% endhighlight %}
